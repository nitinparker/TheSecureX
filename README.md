# TheSecureX

TheSecureX is a monorepo for multiple interlinked applications focusing on machine learning, AI, and data analysis.

## Structure

- **apps/**: Contains the main application code.
- **libs/**: Shared libraries and utilities.
- **data/**: Data files for analysis and training.
- **notebooks/**: Jupyter notebooks for experimentation and analysis.
- **scripts/**: Utility scripts for automation and maintenance.
